# Lessons Learned

## 2026-04-05

- No depender de un solo modelo preview para la generación visual.
- Cualquier export web real necesita un endpoint de descarga en backend, no solo un `alert`.
- Mantener el costo visible desde el inicio ayuda a evitar flujos demasiado caros.
- Para mejorar consistencia visual, un `style bible` compartido funciona mejor que prompts sueltos por escena.
- Si el proyecto usa Veo, conviene separar el modo premium del modo ahorro para no romper el presupuesto.
- Las descargas de assets deben vivir con TTL y limpieza automática para no acumular archivos.
